Predictive Analysis of Type-2 Diabetes using Machine Learning
Project Overview

This project focuses on predicting the risk of Type-2 diabetes using Machine Learning techniques. The system analyzes medical, lifestyle, and socio-economic factors to identify individuals at risk of diabetes and support early healthcare intervention.

Objectives
Predict Type-2 diabetes using machine learning algorithms.
Compare multiple classification models.
Handle class imbalance using SMOTE.
Identify important risk factors influencing diabetes.
Develop a real-time prediction system using Streamlit.
Dataset
Large-scale healthcare dataset
Approximately 253,680 records
22 features including BMI, Age, General Health, Income, and Blood Pressure
Technologies Used
Python

Machine Learning Models
Logistic Regression

Support Vector Machine (SVM)
Random Forest (Final Model)
Methodology
Data Collection
Data Preprocessing
Feature Scaling

Model Training and Evaluation
Real-Time Prediction System
Results
Logistic Regression Accuracy: 73%
Random Forest Accuracy: 83%
Cross-Validation Accuracy: 84%
Random Forest selected as the best-performing model
